# Git Often Used Command

## alias
in fish.config
* `g` <- `git`

## Basic
in .gitconfig
* `g d` diff
* `g co` checkout
* `g ci` commit
* `g st` status
* `g br` branch
* `g ba` branch -a
* `g ps` push origin HEAD
* `g pl` pull origin HEAD
* `g hist` Display the log of commits.
* `g llog` Displays the files that have been changed.
